# Basic-Notes-App
Create new notes using terminal cmd (Basic Introduction to node.js Project)

- To `add` new note:
`node app.js add --title="<title>" --body="<description>"`

- To `remove` existing note:
`node app.js remove --title="<title>"`

- To `list down` all the existing node:
`node app.js list`

- To `read` a particular note description:
`node app.js read --title="<title>"`

NPM packages used:
`chalk`
`yargs`
